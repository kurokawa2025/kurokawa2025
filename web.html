PERSONAL WORK

ポートフォリオサイト

自身のポートフォリオサイトを、デザインから実装ま

で一貫して制作を行なった。HTML・CSSのスキルを伝

える作品として、基礎的なコーディングが反映されて

いることがわかりやすいシンプルなページ作りを意識。

今後は継続的にアップデートを重ね、修正を繰り返し

ながら自分自身のホームベースとして活用していく。

コンセプト

Webデザイナーへの就職活動をするため、自身のデザインへの考

えや作品を紹介できるポートフォリオサイトを制作。

WEBデザイン業務は未経験のため、これまでの作品を通して、”

制作するプロダクトの本質を捉え、その価値を届けるために真摯

にデザインの表現と向き合う姿勢”がシンプルに伝わるように企

画している。

ツール:Illustrator / Photoshop / HTML / CSS / Figma

使用フォント:Noto Sans CJK JP,Open Sans

情報設計

コンテンツを必要最低限に絞り、回遊しやすさを重視した構成に

している。トップページではプロフィールやスキルを簡潔に伝え、

下層ページで私の人となりをより深く知ってもらえる後構成にし

た。スキルセットだけでなく、価値観を伝えることで、企業との

親和性を判断してもらいやすいよう工夫している。作品はジャン

ルごとに代表作を掲載し、今後さらに充実させていく予定。

デザイン

文字情報が多くなるため、シンプルなレイアウトと十分な行間を

意識し、読みやすさを重視している。また、「物事に真摯に向き

合う姿勢」や「明るく元気な人柄」を直感的に表現するために、

赤を基調とした配色にしている。全体的に、親しみやすさを感じ

てもらえるデザインを目指した。
